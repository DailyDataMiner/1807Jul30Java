Collection

(the root interface of collection hierarchy)

public interface Collection<E>

extends Iterable<E>

E are the type of elements in this collection

The JDK does not provide any direct implementation of this interface.
Rather, it provides group of classes (i.e. ArrayList, Hashset)
and subinterfaces (i.e. List, Queue) that make up the collections framework.
Depending on the task in hand, some collections are better for
manipulating a group of elements than others.
