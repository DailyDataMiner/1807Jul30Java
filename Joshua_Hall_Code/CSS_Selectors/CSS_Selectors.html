CSS Selectors


Simple CSS Selectors

  • Type Selector
    • Applies to all elements of the given type
    • Syntax is typename
  • Class Selector
    • Applies to all elements of the selected class
    • Syntax is .classname
  • ID Selector
    • Applies to the element with the given ID
    • Syntax is #id
  • Universal Selector
    • Applies to all elements
    • Can optionally be restricted to a specific namespace
    • Namespaces aren't really a thing is HTML, but they do exist in XML
    • Syntax is *

Attribute Selectors

  • Applies to elements with the given attribute
  • There are multiple valid syntaxes depending on what you want to do
  • Syntax [attr] applies to all elements with the given attribute
  • Syntax [attr=value] applies to elements where the given attribute has the given value
  • Syntax [attr~=val] applies to elements where the value of the given attribute contains the word "val"
  • Syntax [attr*=val] applies to elements where the value of the given attribute contains the substring "val"
  • Syntax [attr^=val] applies to elements where the value of the given attribute starts with "val"
  • Syntax [attr$=val] applies to elements where the value of the given attribute ends with "val"

Combinators

  • General Sibling Combinator
    • Syntax is type1 ~ type2
    • Applies to all elements of type2 that come after an element of type1
  • Adjacent Sibling Combinator
    • Syntax is type1 + type2
    • Applies to all elements of type2 that come immediately after an element of type1
  • Child Combinator
    • Syntax is type1 > type2
    • Applies to all type2 elements that are the child of an element of type1
  • Descendant Combinator
    • Syntax is type1 type2
    • Applies to all type2 elements that are inside of an element of type1
