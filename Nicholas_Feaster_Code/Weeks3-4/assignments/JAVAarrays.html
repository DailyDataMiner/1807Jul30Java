Arrays

  • The JAVA language contains an array class which is used to create Array objects. The class is public and final. The fact that it is final means that no class can extend the Array class. As a result, there are no subclasses of the JAVA standard Array class. However, the Array class is a subclass of the class Object, which means the Array class can use the methods of the Object class.

  • Some of the methods from the Object class are
    1. clone()
    2. equals()
    3. finalize()
    4. getClass()
    5. hashCode()
    6. toString()
  • Method Description
    clone() returns a copy of the array
    equals() returns a boolean that indicates whether or not the arrays are the same
    finalize() Called by the garbage collector on an object when garbage collection determines that there are no more references to the object.
    getClass() returns the class of the object at runtime
    hashCode() returns a hash code value for an object
    toString() returns a string consisting of the name of the class of which the object is an instance, the at-sign character `@', and the unsigned hexadecimal representation of the hash code of the object

    • The toString and equals methods are many times overriden.

  • Conceptually, JAVA arrays are a grouping of zero or more objects of a single type. JAVA arrays are indexed so that each object in the array has an associated non-negative number that can be used to return the object from the array. JAVA arrays can hold duplicates meaning that a single array can hold multiple idential objects of the same type. The type of object a JAVA array holds is determined when it is intilaized.

  • A JAVA array is initilized by typing the name of the object (or primitive) type, that the array is to hold, followed by square brackets, a space, and then a name for the array. After the name an equal sign follows , on the right of the equal side are curly braces. Within the curly braces may be either a length for the array, the object which the array is to hold.

  • JAVA contains multidimensional arrays. In JAVA a multidimensional array is an array of arrays. JAVA arrays can have length zero. One of the most important non-inherited methods of the Array class is the length method. An array's length method returns the length of the arrray as an int value. The last index of an array is one less than its length.

  • Arrays also have getters and setters, which can be used to access and modify array values. Since arrays only contain values of a specific type the getter and setter methods reflect which type would be returned or modified. For example,

  • Method Description
    getInt(Object array, int index) Returns the value of the indexed component in the specified array object, as an int.
    setInt(Object array, int index, int i) Sets the value of the indexed component of the specified array object to the specified int value.
