STRUCTURED QUERY LANGUAGE

Designed for managing data held in a Relational Database Mangement System (RDBMS). SQL statements are used to perform tasks such as update data on a database, or retrieve data from a database. Some common relational database management systems that use SQL are: Oracle, Sybase, Microsoft SQL Server, Access, Ingres, etc. Although most database systems use SQL, most of them also have their own additional proprietary extensions that are usually only used on their system. However, the standard SQL commands such as "Select", "Insert", "Update", "Delete", "Create", and "Drop" can be used to accomplish almost everything that one needs to do with a database. These commands belong to several SQL sublanguages which will be covered below.

Allows a database user to create and restructure database objects, such as the creation or the deletion of a table

  • CREATE 

    CREATE TABLE table_name(
    column1 datatype,
    column2 datatype,
    column3 datatype,
    .....
    columnN datatype,
    PRIMARY KEY( one or more columns )
);

  • ALTER 

    ALTER TABLE table_name
ADD column_name datatype;

  • DROP 

    DROP TABLE table_name;

  • TRUNCATE 

    TRUNCATE TABLE table_name;
Manipulate data within objects of a relational database

  • INSERT 

    INSERT INTO TABLE_NAME (column1, column2, column3,...columnN)  
VALUES (value1, value2, value3,...valueN);

  • UPDATE 

    UPDATE table_name
SET column1 = value1, column2 = value2...., columnN = valueN
WHERE [condition];

  • DELETE 

    DELETE FROM table_name
WHERE [condition];
Create objects related to user access and also control the distribution of privileges among users

  • GRANT 

    GRANT privilege-type ON [TABLE] { table-Name | view-Name } TO grantees

  • REVOKE 

    REVOKE privilege-type ON [ TABLE ] { table-Name | view-Name } FROM grantees
Allow the user to manage database transactions

  • COMMIT 

    INSERT INTO regions VALUES (5, 'Antarctica'); 
COMMIT
    COMMENT 'example comment here';

  • ROLLBACK 

    ROLLBACK TO SAVEPOINT sp1;

  • SAVEPOINT 

    UPDATE employees 
    SET salary = 7000 
    WHERE last_name = 'Banda';
SAVEPOINT banda_sal;

UPDATE employees 
    SET salary = 12000 
    WHERE last_name = 'Greene';
SAVEPOINT greene_sal;
Retrieve data from SQL server

  • SELECT 

    SELECT * 
    FROM employees 
    WHERE department_id = 30
    ORDER BY last_name;
