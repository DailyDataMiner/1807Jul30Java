Collections



What are 'Collections'?

Not 'what are collections?', but rather, what is collections?

'Collections' is an class in Java which is the foundation upon which the collections framework is built. It declares the core methods that all collections will have.

Some methods:

  • boolean add(Object obj)
    • This method takes in an object as a parameter and adds it to a collection.

  • boolean contains(Object obj)
    • This method sees if an object is currently in a list, and if so, returns true.

  • boolean remove(Object obj)
    • This method will remove an object in a list, and if it does so successfully, returns true.

  • int size( )
    • This method returns the size of a collection, aka how many objects it contains.


Example Code Snippet

public static void main(String[] args) {
     ArrayList nums = new ArrayList<>();
     nums.add(3);
     nums.add(1);
     nums.add(2);
     Collections.sort(nums);
     System.out.println(nums.get(0)); // Prints 1
     System.out.println(Collections.max(nums)); // Prints 3
}

As you can see, this code snippet takes advantage of our methods from the Collections class. First it creates a new List of Integers then uses the .add() method to populate the list.

The sort method is then called from the class to sort the array, while the get method is then used to retrieve the integers from the now sorted array. Lastly, we have the .max() method returning the highest integer in the array.
