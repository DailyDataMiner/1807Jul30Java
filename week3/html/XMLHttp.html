XMLHTTP Request Objects

XMLhttp request objects are objects used to exchange data with a server.
Concept Meaning/Use
Var xhr=new XMLHTTPrequest(); This is the syntax used to create a new XML request object.
abort(); This is a function called to cancel the current XMLHTTP request.
async=true(or false); Determines wether to execute the request asynchronously or not
GetAllResponseHeaders() this function returns the metadata of the request
open(method, url, async, user, pw) specifies a request
send()/send(string) sends a request to the server
setRequestHeader() adds a label/value pair to the header to be sent
readystate this holds the status of the XRH
onreadyStatechange defines function to be executed when the ready state property changes.
States
  • 0:request not initialized
  • 1:server connection established
  • 2:Request Recieved
  • 3:server proccessing request
  • 4:response is ready
AJAX workflow
  1. create XMLHTTPRequest object
  2. define onreadystatechange function
  3. Open request
  4. send request
