Introduction


Lambda Expressions

Lambda expressions are anonymous methods which implement the single abstract method of a functional interface. They enable you to pass functionality as an argument to another method. Lambdas are functions, meaning they do not belong to any class. Lambda expressions are an example of how Java is becoming more "functional." Lambda expressions are closures; the body of a lambda expression has the same scope as a nested block. Lambda expressions cannot be generic, but the functional interface associated with them can (e.g. Predicate).




The Predicate Interface

The Predicate interface is a functional interface whose functional method is test(Object). It can be used as the assignment target for a lambda expression. This interface prevents us from having to create numerous new interfaces in order to use lambda expressions.


For example, ArrayList has a removeIf() method which takes a predicate
