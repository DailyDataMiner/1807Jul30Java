What is a Join?

Constructs in SQL that allow you to retreive data from 2 or more tables based on the relationships established between them.






Inner - Will return all records that appear in the left table AND the right table that match based on the join condition.

Full Outer - Will return all of the rows that match based onthe join, as well as those that don't

Right Join - Returns all records that match the join condition as well as all of the records on the right table

Left Join - Returns all records that match the join condition as well as all of the records on the left table that do not.

Self Join - Joining a table with itself(there is no self join keyword)

Cross - Cartesian product between 2 tables, where every row from the first table is joined with every row on the second table

Natural - Inner join where the tables are joined on common column names

Equi Join - Inner join where you use = in the join condition

Non Equi Join - Inner join where you use any other relational operator
